Build:
  
  mvn clean install
	  
Dry run of release:

  mvn release:prepare -DdryRun=true -DautoVersionSubmodules=true
  mvn release:clean
  
Do release:
  
  mvn release:prepare -DautoVersionSubmodules=true
  mvn release:perform
  
Or

Do everything at once

  mvn clean install release:prepare release:perform -DautoVersionSubmodules=true
