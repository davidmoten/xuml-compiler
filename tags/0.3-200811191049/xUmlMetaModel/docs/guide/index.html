How to build this project

If you change the model/uml.ecore that is the underlying metamodel for the xUmlCompiler then you need to regenerate the source from this new ecore file. This is what you do:
  1. Delete model/uml.genmodel
  2. Select model/uml.ecore in Package Explorer
  3. Click right mouse and select New - Other - Eclipse Modelling Framework - EMF Model
